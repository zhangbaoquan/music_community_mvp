# 亲亲心情笔记 — 产品需求文档（PRD v2.0）

> 版本：v2.0 | 状态：规划中 | 最后更新：2026-04-06

---

## 1. 优化目标

v2.0 的核心目标是**解决 v1.0 遗留的技术债务，并横向扩展业务能力**。在保持现有功能稳定的前提下，按优先级渐进式推进。

---

## 2. 优化项（技术债务清偿）

### 🚨 技术债务清偿说明

> **注意：** 所有关于 v1.0 遗留的技术债务与底层优化（包括资源管道化压缩、列表游标分页重构、Postgres RPC 聚合下放、Safari 音视频策略兼容，以及导航架构重构）已整体升级并迁移至 **v3.0 版本**。
>
> 详情请查阅后续的 `prd-v3.0.md` 与对应的开发、测试清单。v2.0 将纯粹聚焦于以下业务新功能的横向扩展。

---

## 3. 新增功能拓展说明

> **注意：** 原定于 v2.0 及后续版本的业务新增功能（包括 1v1 灵魂私信、AI 情绪调音师、个人主页自定义、多端 Native App 适配等），现已**全部统一入库至需求池（Backlog）**。
>
> 今后的版本迭代中，新功能将直接从需求池中提取和评估。请查阅 `docs/prd/需求池-Backlog.md` 获取完整的远期功能规划。

---

## 4. 结论

v2.0 阶段作为一个承上启下的版本，其核心历史使命已经完成（主要为线上稳定性治理、巨型文件拆解与代码质量提升）。
更深水区的技术底座重构已转入 v3.0 进行；而业务创新探索则移步需求池，等待后续版本发力。
